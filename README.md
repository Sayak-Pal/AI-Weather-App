AI Weather & Chat AppThis is a simple, responsive web application that displays the weather and includes an AI-powered chatbot that can answer questions based on the live weather data.FeaturesLive Weather Data: Get up-to-the-minute weather for any location.Search by City: Type in any city name to see its current weather.Current Location: Click the location icon to get the weather for your current position (requires browser permission).Animated Background: The app features a beautiful, cycling gradient background that transitions from day to night.Context-Aware AI Chat: Open the chat and ask the AI questions. It uses the currently displayed weather as context for its answers.Fully Responsive: Designed to look great on both mobile devices and desktop screens.ðŸš€ Setup & InstallationTo run this application, you only need a modern web browser.
